# 📊 Rekap Nilai PKM dan Rekomendasi Reviewer Fakultas

## 📌 Tentang Proyek

Proyek ini dikembangkan untuk membantu proses rekapitulasi dan analisis hasil penilaian Program Kreativitas Mahasiswa (PKM) Tahun 2025. Sistem ini dirancang untuk mengotomatisasi pengolahan data penilaian serta memberikan rekomendasi reviewer berdasarkan hasil evaluasi proposal.

---

## 🎯 Tujuan Proyek

* Mengotomatisasi proses rekapitulasi nilai proposal PKM.
* Menyediakan informasi hasil penilaian.
* Membantu proses pengambilan keputusan terkait rekomendasi reviewer.
* Menyajikan hasil analisis dalam bentuk dashboard yang mudah dipahami.

---

## 📂 Dataset

Dataset yang digunakan merupakan data penilaian proposal PKM yang mencakup informasi mengenai:

* Identitas proposal
* Fakultas pengusul
* Skema PKM
* Nilai reviewer
* Hasil rekapitulasi nilai
* Rekomendasi reviewer

**Catatan:** Seluruh data yang dipublikasikan pada repository ini telah dianonimkan untuk menjaga kerahasiaan dan privasi peserta.

---

## 🔒 Privasi dan Keamanan Data

Repository ini dipublikasikan untuk tujuan portofolio dan pembelajaran. Oleh karena itu, seluruh informasi pribadi telah dianonimkan sebelum dipublikasikan.

Data yang dianonimkan meliputi:

* Nama peserta
* Nomor Induk Mahasiswa (NIM)
* Nomor telepon
* Alamat email

Proses anonimisasi dilakukan untuk melindungi informasi pribadi serta menerapkan prinsip etika dalam pengelolaan data.

## 🔍 Tahapan Pengerjaan

### 1. Data Cleaning

* Memastikan konsistensi format data.
* Memvalidasi data yang tidak lengkap.
* Menghapus data yang tidak relevan untuk kebutuhan analisis.

### 2. Data Processing

* Mengolah nilai proposal.
* Menghitung hasil rekapitulasi nilai.
* Menentukan rekomendasi reviewer berdasarkan kriteria yang ditetapkan.
* Menerapkan conditional formatting untuk menyoroti kategori nilai dan mempermudah proses evaluasi data.

### 3. Data Analysis

Analisis dilakukan untuk mengetahui:

* Distribusi proposal berdasarkan fakultas.
* Distribusi proposal berdasarkan skema PKM.
* Sebaran hasil penilaian proposal.
* Rekomendasi reviewer berdasarkan hasil evaluasi.

### 4. Dashboard Development

Dashboard dikembangkan menggunakan Microsoft Excel untuk menampilkan:

* Jumlah proposal.
* Distribusi proposal per fakultas.
* Distribusi proposal per skema PKM.
* Ringkasan hasil penilaian.
* Informasi rekomendasi reviewer.
